Git file directory:
1) Go to directory <path>/tutorials/tutorials_app/run

2) in file run.sh chage variables:
    a) RUN_DIR (full path to directory RUN)
	b) GIT_REPO (GIT repository path)
	
3) configure CRON to run run.sh file
  a) In terminal, from root user run
  # crontab -e
  
  This will open text editor to modify crontab
  
  b) In editor add line like this for hourly updating:
  
0 * * * * bash /home/tutorials/tutorials_app/run/run.sh
or for daily update at 12:00 pm
0 0 * * * bash /home/tutorials/tutorials_app/run/run.sh
