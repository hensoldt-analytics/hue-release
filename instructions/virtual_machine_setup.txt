1) Download and install Vagrant (http://downloads.vagrantup.com/tags/v1.1.0)
2) Download basic box and (you would need to do it only once):
https://docs.google.com/a/hortonworks.com/file/d/0BzuJsGNSrhIAa01aU0czQi1DbEk/edit?usp=sharing
3) vagrant box add sandbox package.box
4) git clone git@github.com:hortonworks/sandbox-shared.git
cd sandbox-shared
git checkout Caterpillar
cd deploy/vagrant
5) vagrant plugin install vagrant-vbguest
6) vagrant up (first time it would take some time to build VM).
7) To connect to running VM simply run vagrant ssh
